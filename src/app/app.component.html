Country pipes

The examples written for:

Country name: Israel

Alpha 2 (iso_3166_1_alpha2): IL

Country Anthem - a2ToAnthem

IL | a2ToAnthem
Result: 
{{ 'IL' | a2ToAnthem }}

Country name - a2ToCountry

Default: common name

IL | a2ToCountry
Result: 
{{ 'IL' | a2ToCountry }}

Official name

IL | a2ToCountry: [ 'official' ]
Result: 
{{ 'IL' | a2ToCountry: [ 'official' ] }}

Native name - Default

IL | a2ToCountry: [ 'native' ]
Result: 
{{ 'IL' | a2ToCountry: [ 'native' ] }}

Native name - Specific language - by ISO 639-3 language code

IL | a2ToCountry: [ 'native', 'ara' ]
Result: 
{{ 'IL' | a2ToCountry: [ 'native', 'ara' ] }}
